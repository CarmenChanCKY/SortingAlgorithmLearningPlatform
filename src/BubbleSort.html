Method

Ascending Order:

  1. Prepare a list of unsorted elements A[0...n-1], while n is the number of elements
  2. Start from A[i] where i = 0, compare A[i] to A[i+1]
  3. If A[i] is larger than A[i+1], swap the position of A[i] and A[i+1]
  4. Move to the next element A[i+1] and repeat step 3 until A[i] is smaller than A[i+1]
  5. Repeat step 2-4 where i = 1 to n-2
  6. If the list is still unsorted, repeat step 2-5 and start from i = 0 again. Stop it until the list is sorted

Descending Order:

  1. Prepare a list of unsorted elements A[0...n-1], while n is the number of elements
  2. Start from A[i] where i = 0, compare A[i] to A[i+1]
  3. If A[i] is smaller than A[i+1], swap the position of A[i] and A[i+1]
  4. Move to the next element A[i+1] and repeat step 3 until A[i] is larger than A[i+1]
  5. Repeat step 2-4 where i = 1 to n-2
  6. If the list is still unsorted, repeat step 2-5 and start from i = 0 again. Stop it until the list is sorted

Algorithm

Ascending Order: 

                    for (int i = arr.length - 1; i > 0; i--) {
                        for (int j = 0; j < i; j++) {
                            if (arr[j] >= arr[j + 1]) {
                                int temp = arr[j];
                                arr[j] = arr[j + 1];
                                arr[j + 1] = temp;
                            }
                        }
                    }

Early-termination & ascending order: 

                    boolean swap = false;
                    for (int i = arr.length - 1; i > 0; i--) {
                        swap = false;
                        for (int j = 0; j < i; j++) {
                            if (arr[j] >= arr[j + 1]) {
                                swap = true;
                                int temp = arr[j];
                                arr[j] = arr[j + 1];
                                arr[j + 1] = temp;
                            }
                        }

                        }
                    }

Example

For ascending order:
There are a list of unsorted elements A[0...4]

  1. Start from A[0], compare A[0] to A[1]
    2. Because A[0] is smaller than A[1], so we do nothing in this case
  2. Compare A[1] to A[2]
    3. In this time, A[1] is larger than A[2]. So we swap the position of A[1] and A[2]
  3. After swaping, A[2] is smaller than A[3]. So we do nothing in this case
  4. Compare A[3] to A[4]
    5. A[3] is larger than A[4]. So we swap the position of A[3] and A[4]
  5. The list is still unsorted. So we compare A[0] to A[1] again
  6. A[0] is larger than A[1]. So we swap the position of A[0] and A[2]
  7. After checking the whole list, this list has been sorted in ascending order

For descending order:
There are a list of unsorted elements A[0...4]

  1. Start from A[0], compare A[0] to A[1]
    2. Because A[0] is smaller than A[1], so we swap the position of A[0] and A[1].
  2. After swaping, A[1] is still smaller than A[2].
    3. So we also swap the position of A[1] and A[2].
  3. Compare A[2] to A[3]
    4. A[2] is larger than A[3]. So we still do nothing in this case
  4. Compare A[3] to A[4]
    5. A[3] is smaller than A[4]. So we swap the position of A[3] and A[4]
  5. The list is still unsorted.
  6. So we compare A[0] to A[1] again
    7. A[0] is smaller than A[1]. So we swap the position of A[0] and A[1]
  7. Compare A[1] to A[2]
    8. A[1] is larger than A[2]. So we do nothing in this case
  8. Compare A[2] to A[3]
    9. A[2] is smaller than A[3]. So we swap the position of A[2] and A[3]
  9. Compare A[3] to A[4]
    10. A[3] is larger than A[4]. So we do nothing in this case
  10. The list is still unsorted.
  11. So we compare A[0] to A[1] again
    12. A[0] is larger than A[1]. So we do nothing in this case
  12. Compare A[1] to A[2]
    13. A[1] is smaller than A[2]. So we swap the position of A[1] and A[2]
  13. After checking the whole list, this list has been sorted in descending order
