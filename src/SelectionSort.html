Method

Ascending Order:

  1. Prepare a list of unsorted elements A[0...n-1], while n is the number of elements
  2. Start from A[i] where i = 0, compare A[i] to A[i+1...n-1]
  3. If any element inside A[i+1...n-1] is smaller than A[i+1], and it is the smallest element inside A[i+1...n-1], mark the position of this element as x
  4. Swap the position of A[i] and x
  5. Repeat step 2-4 where i = 1 to n-2
  6. Stop it until the list is sorted

Descending Order:

  1. Prepare a list of unsorted elements A[0...n-1], while n is the number of elements
  2. Start from A[i] where i = 0, compare A[i] to A[i+1...n-1]
  3. If any element inside A[i+1...n-1] is larger than A[i+1], and it is the largest element inside A[i+1...n-1], mark the position of this element as x
  4. Swap the position of A[i] and x
  5. Repeat step 2-4 where i = 1 to n-2
  6. Stop it until the list is sorted

Algorithm

Ascending Order: 

                    for (int i = 0; i < arr.length - 1; i++) {
                        int smallest = i + 1;
                        for (int j = i + 1; j <= arr.length - 1; j++) {
                            if (arr[smallest] >= arr[j]) {
                                smallest = j;
                            }
                        }
                        if (arr[i] >= arr[smallest]) {
                            int temp = arr[smallest];
                            arr[smallest] = arr[i];
                            arr[i] = temp;
                        }
                    }

Example

For ascending order:
There are a list of unsorted elements A[0...4]

  1. Start from A[0], compare A[0] to A[1...4]
    2. Because A[1] is smaller than A[0], and A[1] is the smallest element inside A[1...4], so we swap the position of A[0] and A[1]
  2. Compare A[1] to A[2...4]
    3. A[2] is larger than A[1], and A[2] is the smallest element inside A[2...4], so we swap the position of A[1] and A[2]
  3. A[2] is smaller than both A[3] and A[4]. So we do nothing in this case
  4. Compare A[3] to A[4]
    5. A[3] is larger than A[4]. So we swap the position of A[3] and A[4]
  5. The list is sorted in ascending order

For descending order:
There are a list of unsorted elements A[0...4]

  1. Start from A[0], compare A[0] to A[1...4]
    2. Because A[4] is larger than A[0], and A[1] is the largest element inside A[1...4], so we swap the position of A[0] and A[4]
  2. Compare A[1] to A[2...4]
    3. In this time, A[2] is larger than A[1], and A[2] is the largest element inside A[2...4], so we swap the position of A[1] and A[2]
  3. Compare A[2] to A[3...4]
    4. A[4] is larger than A[2], and A[4] is the largest element inside A[3...4], so we swap the position of A[4] and A[2]
  4. Compare A[3] to A[4]
    5. A[4] is larger than A[3]. So we swap the position of A[3] and A[4]
  5. After checking the whole list, this list has been sorted in descending order
