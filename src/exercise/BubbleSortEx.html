Q5. Which answer(s) about Insertion Sort is correct?
A. Early-termination will occur when there are no swapping in the whole list
B. Comparison will start at the first element
C. Number of loops is needed to sort a list is depends on list size
D. By using bubble sort, sort in ascending order is faster than sort in descending order.

Explanation:

For answer B, the time needed to sort a list depends on the order of elements inside the list.
For example, if the elements inside a list is nearly sorted in ascending order such as 1 2 3 9 5, sort in ascending order is faster than sort in descending order.
But, if if the elements inside a list is nearly sorted in ascending order such as 9 7 5 0 8, sort in descending order is faster than sort in ascending order.

For answer C, number of loops is needed to sort a list is not related to element size.
For example, to sort 9 1 3 8 4 in ascending order needs at least 8 loops.
Sorting 5 1 7 3 9 in descending order needs at least 13 loops.
Sorting 5 0 6 9 2 in descending order needs at least 12 loops.
This three examples have 5 elements in the list, but they needs different number of loops to sort them. So, C is incorrect.
